#Smart Resume Parser

**Resume Analysis App** - Extract skills, emails, education from PDF/DOCX!

##  Features
- ✅ Upload multiple resumes
- ✅ Extract Python, React, ML skills (30+)
- ✅ Email & phone detection
- ✅ Skill search ("py" → Python)
- ✅ Analytics dashboard
- ✅ CSV/JSON export
## How to run the app


This will open the app in your browser at:

http://localhost:8501
